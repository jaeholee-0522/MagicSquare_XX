## 6. 기존 구현 확인 (변경 없음)

본 세션에서 수정 허용 범위로 명시된 파일명(`schemas.py`, `input_validator.py`)과 달리, 저장소는 **Report/09 GREEN** 이후 아래 모듈을 사용 중이다.

| 실제 파일 | Layer | 역할 |
|---|---|---|
| `src/boundary/contracts.py` | Boundary | `ValidationErrorResponse`, `INVALID_SIZE_*` |
| `src/boundary/boundary_validator.py` | Boundary | `BoundaryValidator.validate()` — `grid is None` → `invalid_size()` |
| `src/boundary/__init__.py` | Boundary | 패키지 docstring |

**핵심 분기 (기존):**

```text
BoundaryValidator.validate(grid)
  → _is_size_invalid(grid)
      → grid is None → True
  → ValidationErrorResponse.invalid_size()
      → code="INVALID_SIZE", message="Grid must be 4x4."
```
