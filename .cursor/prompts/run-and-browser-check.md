# 앱 실행 및 브라우저 확인 프롬프트

현재 개선된 앱을 실행하고 브라우저에서 확인해 주세요.

## 작업
1. 프로젝트 실행 방법을 먼저 확인한다.
2. `package.json`, `README`, 실행 스크립트를 확인한다.
3. 필요한 경우 의존성 설치 여부를 확인한다.
4. 개발 서버를 실행한다.
5. 브라우저에서 주요 화면을 확인한다.
6. 오류가 발생하면 원인과 해결 방법을 보고한다.

## 금지
- 사용자 승인 없이 패키지 버전 대규모 변경 금지
- 임의로 프레임워크 교체 금지
- 배포 금지
- API Key 또는 환경변수 출력 금지

## 출력
# 실행 명령
# 접속 URL
# 확인한 화면
# 발견한 오류
# 해결 여부
